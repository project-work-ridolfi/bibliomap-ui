# Changelog

Tutte le modifiche rilevanti al progetto sono documentate in questo file.

## [1.0.0] - **IN CORSO**

Prima release pubblica di Bibliomap.

### Funzionalità principali

- Sistema di autenticazione con OTP via email
- Gestione librerie personali con visibilità configurabile
- Catalogazione libri con copertine e tag
- Sistema completo di prestiti tra utenti
- Ricerca libri con filtri avanzati
- Dashboard per gestione prestiti attivi e richieste
- Suggerimenti basati su affinità letteraria
- Statistiche personali e globali
- Mappa interattiva delle librerie
- Modalità light e dark
- Design responsive
